Política de Privacidade

DETRAN Simulado
⚠️ VERSÃO DE DEMONSTRAÇÃO: Esta é uma versão de teste do aplicativo DETRAN Simulado. As perguntas contidas no aplicativo são apenas para demonstração e NÃO são questões oficiais do DETRAN.

1. Informações Gerais

Esta Política de Privacidade descreve como coletamos, usamos e protegemos suas informações quando você utiliza o aplicativo DETRAN Simulado. Estamos comprometidos em proteger sua privacidade e garantir a transparência sobre como tratamos seus dados.

Desenvolvedor: Fernando Alencar
Aplicativo: DETRAN Simulado
Última atualização: Dezembro de 2024

Importante: Se o seu público-alvo incluir crianças com menos de 13 anos, consulte a Política de Dados do Utilizador para evitar violações comuns de privacidade infantil.

2. Informações que Coletamos

2.1 Dados Coletados Automaticamente

  • Progresso do Simulado: Resultados dos testes, pontuações e estatísticas de desempenho
  • Preferências do App: Configurações de tema (claro/escuro), tempo de cronômetro
  • Dados de Uso: Frequência de uso, questões mais erradas, tempo gasto no aplicativo

2.2 Dados que NÃO Coletamos

  • Informações pessoais identificáveis (nome, e-mail, telefone)
  • Localização geográfica
  • Contatos do dispositivo
  • Fotos ou arquivos do dispositivo
  • Dados de outros aplicativos

3. Como Usamos Suas Informações

Os dados coletados são utilizados exclusivamente para:

  1. Funcionamento do App: Salvar seu progresso e estatísticas de desempenho
  2. Melhorar a Experiência: Personalizar o conteúdo baseado no seu desempenho
  3. Análise de Uso: Entender como o aplicativo é utilizado para futuras melhorias

4. Armazenamento e Segurança dos Dados

4.1 Armazenamento Local

Todos os seus dados são armazenados localmente no seu dispositivo usando AsyncStorage. Nenhuma informação é enviada para servidores externos.

4.2 Segurança

  • Os dados ficam protegidos pelas medidas de segurança do seu dispositivo
  • Não compartilhamos dados com terceiros
  • Não fazemos backup automático para nuvem

5. Seus Direitos

Você tem os seguintes direitos sobre seus dados:

  • Acesso: Visualizar suas estatísticas e progresso dentro do aplicativo
  • Exclusão: Limpar todos os dados através das configurações do app
  • Portabilidade: Os dados ficam apenas no seu dispositivo

6. Dados de Menores de Idade

O aplicativo DETRAN Simulado é destinado a pessoas que estão se preparando para o exame de direção. Não coletamos intencionalmente dados de menores de 13 anos. Se você é menor de 13 anos, peça para um responsável revisar esta política antes de usar o aplicativo.

7. Alterações nesta Política

Esta Política de Privacidade pode ser atualizada ocasionalmente. Quando isso acontecer:

  • A data de "Última atualização" será modificada
  • As mudanças entrarão em vigor imediatamente
  • Recomendamos revisar esta página periodicamente

8. Permissões do Aplicativo

O aplicativo solicita apenas as seguintes permissões essenciais:

  • INTERNET: Para futuras funcionalidades de atualização (atualmente não utilizada)
  • Armazenamento Local: Para salvar progresso e configurações

9. Uso de Dados Anônimos

Transparência Total: Este aplicativo foi desenvolvido com foco na privacidade. Todos os dados permanecem no seu dispositivo e não são compartilhados com terceiros, anunciantes ou serviços de análise.
